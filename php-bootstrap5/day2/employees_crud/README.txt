คู่มือเริ่มต้นระบบ CRUD พนักงาน (PHP + MySQLi + Bootstrap 5)

1) นำไฟล์ schema.sql ไป Import ใน MySQL เพื่อสร้างฐานข้อมูล employees_db และข้อมูลตัวอย่าง
2) วางไฟล์ทั้งหมดไว้ในโฟลเดอร์เว็บ เช่น htdocs/employees_crud/
3) ปรับค่าการเชื่อมต่อฐานข้อมูลใน db.php ให้ตรงกับเครื่องของคุณ
4) เปิดใช้งานผ่านเบราว์เซอร์: http://localhost/employees_crud/index.php

โครงสร้างไฟล์
- db.php
- index.php
- insert_employee.php
- edit.php
- update_employee.php
- delete_employee.php
- schema.sql
- README.txt
